Mục đích
- Thư mục SRA_doc lưu các báo cáo, tổng hợp văn bản, biên bản họp và các chứng cứ phục vụ phân tích hệ thống mô phỏng.

Hướng dẫn sử dụng
- Bổ sung từng file theo template có sẵn.
- Khi thu thập văn bản luật/quy định: lưu file gốc hoặc link và tóm tắt điểm áp dụng.
- Khi khảo sát hệ thống ngoài: chụp ảnh giao diện, ghi lại luồng thao tác chính và list pain points.

Danh sách file chính (template)
- analysis_report.md — Báo cáo phân tích/đánh giá hệ thống hiện tại
- regulations.md — Tổng hợp văn bản, Thông tư, Nghị định, biểu mẫu liên quan
- process_analysis.md — Phân tích & mô hình quy trình nghiệp vụ (BPMN / flowcharts)
- benchmarking.md — Khảo sát các hệ thống tương tự (so sánh tính năng và pain points)
- brainstorming.md — Ghi chép nhóm tư duy / quyết định tính năng
- meeting_minutes.md — Mẫu biên bản tóm tắt họp/thảo luận

- Thu thập các văn bản pháp lý nếu bài toán liên quan đến tính phí/thuế/giấy phép.
